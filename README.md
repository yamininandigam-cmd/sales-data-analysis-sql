# 📊 Sales Data Analysis (SQL Project)

## 📌 Overview

This project demonstrates data analysis using SQL by building a relational database and generating insights from sales data.

---

## 🎯 Objectives

* Analyze revenue trends
* Identify top-selling products
* Understand customer behavior

---

## 🛠️ Tools Used

* SQL (MySQL / SQL Server)
* Joins, Group By, Subqueries

---

## 📂 Project Structure

* `sales_analysis.sql` → Contains table creation, data insertion, and analysis queries

---

## 📊 Key Analysis

* Total revenue calculation
* Revenue by product category
* Top-selling products
* Customer purchase behavior
* Advanced filtering using subqueries

---

## 🚀 How to Run

1. Create a database
2. Run the SQL script
3. Execute queries to view insights
